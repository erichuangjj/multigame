# Monster Blocks

一个有趣的怪物配对游戏,测试你的记忆力和反应速度!

## 游戏规则

1. 点击"开始游戏"按钮开始
2. 在60秒内找出所有相同的怪物方块
3. 点击两个相同的怪物方块进行配对
4. 每成功配对一次得10分
5. 时间用完或完成所有配对游戏结束

## 操作说明

- 点击方块选择
- 再次点击取消选择
- 选择两个相同的方块进行配对
- 配对成功会获得分数
- 配对失败会取消选择

## 游戏提示

- 仔细观察每个怪物的颜色和特征
- 合理规划配对顺序
- 注意剩余时间
- 保持冷静,不要慌张

## 技术说明

- 使用纯HTML、CSS和JavaScript开发
- 响应式设计,适配各种屏幕尺寸
- 流畅的动画效果
- 本地存储最高分记录
